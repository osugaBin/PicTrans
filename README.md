# PicTrans - 高质量图片调整工具

一个优雅、简洁的在线图片大小调整工具，支持无损压缩和实时预览。

## ✨ 功能特点

- 🖼️ **支持多种格式**：JPG、PNG、WEBP
- 📏 **精确缩放**：1%-200%的缩放范围
- 👀 **实时预览**：拖动滑块即时查看效果
- 📊 **详细信息**：显示原始尺寸、新尺寸和文件大小
- 🎨 **精美界面**：现代化的毛玻璃设计
- 📱 **响应式设计**：完美适配移动端和桌面端
- 🔄 **无损压缩**：保持原文件格式和质量
- ⚡ **高质量渲染**：使用Canvas高质量图像缩放

## 🚀 快速开始

### 在线使用
直接打开 `index.html` 文件即可在浏览器中使用。

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/osugaBin/PicTrans.git

# 进入项目目录
cd PicTrans

# 使用任意HTTP服务器运行
# Python 3
python -m http.server 8000

# Node.js (需要安装 http-server)
npx http-server

# 或者使用Live Server等VSCode扩展
```

然后在浏览器中访问 `http://localhost:8000`

## 🎯 使用方法

1. **上传图片**：点击上传区域选择要调整的图片
2. **调整大小**：拖动滑块选择缩放比例（1%-200%）
3. **查看信息**：实时查看原始尺寸、新尺寸和文件大小
4. **下载保存**：点击"下载图片"按钮保存调整后的图片

## 🌟 特色功能详解

### 无损压缩技术
- 保持原文件格式（JPG保持JPG，PNG保持PNG）
- PNG格式质量100%，JPEG质量95%优化文件大小
- 使用`imageSmoothingQuality: 'high'`确保高质量渲染

### 文件大小计算
- 实时计算原始文件大小
- 动态预估转换后文件大小
- 智能单位转换（B/KB/MB/GB）

### 响应式设计
- 移动端友好的触控操作
- 自适应布局和字体大小
- 优化的移动端性能
